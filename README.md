# Expense Tracker

A simple expense tracking application built with React MERN stack.

## Features
- User login and registration
- Add, edit, and delete expenses
- Filter expenses by category, date, and amount
- View spending analytics with charts
- Export expense reports as PDF
- Beautiful and responsive UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## Installation

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## How to Use
1. Sign up or login to your account
2. Add expenses with category, amount, and date
3. Filter expenses by different criteria
4. View charts to analyze your spending
5. Download PDF reports of your expenses
